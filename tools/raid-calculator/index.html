How to use

  1. Pick a RAID type — greyed-out types just need more drives than you've added.
  2. Set each drive's size, or use Set all to for identical drives. Add or remove bays as needed.
  3. Read the Result: usable capacity, protection overhead, fault tolerance and efficiency. Mixed sizes? SHR / SHR-2 use them best.

Everything runs in your browser and your setup is saved locally for next time.
